Для сборки и запуска программы нужны Maven 3 и java8 (Прописаны в PATH)
1. С помощью консоли перейти в директорию проекта и выполнить команду: mvn clean package
2. В target директории проекта нужно запустить jar с помощью команды java -jar 
3. Запросы:
	GET api/users - Список всех пользователей
	GET api/users/{email} - Получить пользователя по email
	POST api/users - добавить пользователя	
	DELETE api/users/{id} - Удалить пользователя по id
	PUT api/users/{id} - Изменить пользователя по id
4. Основные примеры находяться в "Примеры Curl.png"
